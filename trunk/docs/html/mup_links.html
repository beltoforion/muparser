Software using the muParser library (alphabetically)

Google chart tools The Google chart tools seem to be using muParser for the postprocessing of chart input data. I took this information from their documentation so i don't know the details. I guess its running at their servers to process data submitted by user JavaScripts.
(Platform: Any JavaScript capable browser)
kst A plotter for scientific data. (currently muParser is used by the Marquardt Levenberg plugin)
(Platform: Linux / Operating systems that support KDE)
MatriX DS4 produces laser systems for industry applications like engraving, cutting, incision and welding. muParser is used in their MatriX application, a software for for managing the laser systems.
(Platform: Linux)
Meshlab    An open source, portable, and extensible system for the processing and editing of unstructured 3D triangular meshes.
(Platform: MS Windows, Linux, OSX)
Pasimodo Pasimodo is a scientific software package for particle based simulations.
(Platform: Linux/BSD)
Transition Maker 2    Creates transition sequences for video processing.
(Platform: MS Windows)
Trininaut A Program launcher with extra features.
(Platform: WinXP, Win2K)
Qtiplot A free platform independent clone of Origin
(Platform: MS Windows / Linux / Mac)
